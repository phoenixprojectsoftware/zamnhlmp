These files define entity placements for the CTF gamemode, effectively making
any map you want supported for Capture the Flag.
For the red team, use the entities 'item_flag_team2' and 'info_player_team2'.
For the blue team, use the entities 'item_flag_team1' and 'info_player_team1'.
You can get your current coordinates in-game by typing 'status' in the console.
